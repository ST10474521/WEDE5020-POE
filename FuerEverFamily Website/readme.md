# FurEver Family Website

A responsive website for FurEver Family pet adoption organization.

## Project Structure

- `index.html` - Homepage
- `About.html` - About us page with organization information
- `Pets.html` - Available pets gallery with lightbox
- `Donation.html` - Donation page with payment form
- `Contact.html` - Contact form and information
- `Volunteer.html` - Volunteer application page (NEW)
- `SuccessStories.html` - Adoption success stories showcase (NEW)
- `style.css` - Main stylesheet
- `script.js` - JavaScript functionality
- `sitemap.xml` - SEO sitemap
- `robots.txt` - Search engine instructions

## Features Implemented

### JavaScript Functionality
- Interactive accordion (About page)
- Gallery lightbox (Pets page)
- Form validation with error messages
- Interactive maps using Leaflet.js
- Search and filter functionality
- Modal windows for adoption inquiries
- Dynamic donation amount selection
- Volunteer form handling (NEW)

### SEO Optimization
- Meta descriptions on all pages
- Semantic HTML structure
- Clean URL structure
- Internal linking
- Mobile-responsive design
- Sitemap.xml and robots.txt

### Form Handling
- Contact form with validation
- Adoption inquiry form
- Donation form with amount selection
- Volunteer application form (NEW)
- Client-side validation with error messages
